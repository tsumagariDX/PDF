# 📄 Modern PDF Utility - Complete Edition

すべての改善が適用された完全版

## ✅ すべての改善点

### 機能改善
- ✅ すべてのタブでドラッグ＆ドロップ対応
- ✅ 圧縮・パスワード設定にデフォルトファイル名設定
- ✅ 抽出・削除機能の修正（関数呼び出しエラー解消）
- ✅ 並び替え・回転機能の修正（関数呼び出しエラー解消）
- ✅ Word変換機能の修正（関数呼び出しエラー解消）
- ✅ パスワード設定時、権限設定が1つでも必須
- ✅ パスワード設定から注釈禁止・フォーム入力禁止を削除（シンプル化）
- ✅ 閲覧パスワード設定時、自動的にコピー・印刷も禁止

### UI改善
- ✅ ウィンドウサイズ拡大（1400x950）で見切れ解消
- ✅ 最小サイズ設定（1200x850）
- ✅ メニューバー追加（ファイル、ヘルプ）
- ✅ 終了ボタン追加
- ✅ 選択中タブの視認性向上（青背景・白文字）
- ✅ タブを全幅に均等配置
- ✅ 圧縮レベルスライダー改善（1-5すべて表示）
- ✅ すべてのタブで統一されたUI

## 🚀 使い方

### インストール
```bash
pip install -r requirements.txt
# オプション（DnD機能用）
pip install tkinterdnd2
```

### 起動
```bash
python main.py
```

## 📋 機能一覧

### 📑 PDF結合
- 複数PDFを1つに統合
- ドラッグ＆ドロップ対応
- リスト内で並び替え可能

### ✂️ ページ抽出／削除
- ページ範囲指定（例: 1,3,5-7）
- ビジュアルプレビュー
- 抽出モード／削除モード

### 🔄 並び替え／回転
- ドラッグでページ並び替え
- 90°/180°回転
- プレビュー表示

### 🗜️ PDF圧縮
- Ghostscript使用
- 5段階の圧縮レベル（改善版スライダー）
- 目標サイズ指定可能

### 📝 変換
- PDFからWord/Excel
- テキスト・表を自動抽出

### 🔒 パスワード保護
- パスワード設定/解除
- シンプルな権限制御:
  - コピー禁止
  - 印刷禁止
  - 閲覧パスワード必須（自動的にコピー・印刷も禁止）
